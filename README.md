# IZO-Indywidualne Zaopatrzenie Ortopedyczne

Responsywna, semantyczna strona wizytówka firmy IZO-Indywidualne Zaopatrzenie Ortopedyczne.

## Uruchomienie

Stronę można otworzyć bezpośrednio przez plik `index.html` albo uruchomić lokalny serwer:

```bash
python3 -m http.server 8000
```

Następnie wejdź na `http://localhost:8000`.

## Przed publikacją

W plikach `index.html`, `robots.txt` i `sitemap.xml` zamień `https://example.com/` na docelową domenę. W `index.html` uzupełnij też prawdziwy telefon, e-mail i godziny pracy. Miejsca wymagające danych firmy są oznaczone słowem `UZUPEŁNIJ`. Gotowa grafika Open Graph znajduje się w `assets/og-cover.jpg`.
